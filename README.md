## 🚀 Proyectos

### 🏠 Airbnb Clone — Plataforma de alquiler vacacional
> Clon funcional de Airbnb construido con NestJS. El proyecto más completo de mi portfolio.

**Stack:** NestJS · TypeScript · TypeORM · PostgreSQL · JWT · Guards · Next.js · Docker

- Auth completo con **JWT + Guards + roles** (host / guest)
- CRUD de propiedades, reservas con lógica de disponibilidad real
- Subida de imágenes, filtros de búsqueda y paginación
- Arquitectura modular en capas limpia y escalable
- Contenedorizado con **Docker Compose**

### 🧠 Taskly — Gestor de tareas con IA
> Kanban en tiempo real con priorización automática por inteligencia artificial

**Stack:** Java 21 · Spring Boot 3 · Spring AI · Groq · PostgreSQL · Next.js · Pusher · Docker

- IA con **Groq (llama-3.1)** prioriza tareas del 1-10 con justificación en español
- Procesamiento **asíncrono** — la IA corre en background sin bloquear al usuario
- **Kanban drag & drop** (dnd-kit) sincronizado en tiempo real con Pusher
- Auth con **Spring Security + JWT** y canales privados
- Deploy completo: **Render + Vercel + Neon**

### 💊 FarmaApp — Sistema de gestión de farmacia
> API REST para inventario, ventas y control de stock con roles

**Stack:** Java 21 · Spring Boot 3 · Spring Security · JWT · PostgreSQL · Docker

- Auth **stateless JWT** con filtro `JwtAuthenticationFilter` personalizado
- Roles (ADMIN / USER) con `@PreAuthorize` en cada endpoint
- Módulo de ventas con descuento automático de stock e historial inmutable
- Modelo relacional optimizado con **JOIN FETCH** — sin problema N+1

[![Repo](https://img.shields.io/badge/Repositorio-GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/whoysAndre/farmaback-uni)
